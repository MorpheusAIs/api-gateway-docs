---
title: "Create API Key"
api: "POST /api/v1/auth/keys"
description: "Create a new API key for the current user"
---

Create a new API key for the current user.

Requires JWT Bearer authentication with the token received from the login endpoint. The full API key is only returned once during creation - store it securely.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token (JWT) from OAuth2 login
</ParamField>

### Body

<ParamField body="name" type="string">
  Optional name for the API key to help identify its purpose
</ParamField>

### Response

<ResponseField name="key" type="string">
  The full API key (only shown once)
</ResponseField>

<ResponseField name="key_prefix" type="string">
  First few characters of the API key
</ResponseField>

<ResponseField name="name" type="string">
  The name of the API key
</ResponseField>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api.mor.org/api/v1/auth/keys \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Production Key"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.mor.org/api/v1/auth/keys', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Production Key'
  })
});

const apiKey = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.mor.org/api/v1/auth/keys',
    headers={
        'Authorization': 'Bearer YOUR_JWT_TOKEN',
        'Content-Type': 'application/json'
    },
    json={'name': 'Production Key'}
)

api_key = response.json()
```
</RequestExample>
