---
title: "Delete User Account"
api: "DELETE /api/v1/auth/register"
description: "Delete the current user's account and all associated data"
---

Delete the current user's account and all associated data.

This action is irreversible and will permanently delete all data associated with your account. Requires JWT Bearer authentication.

### What Gets Deleted

1. All sessions
2. All API keys
3. Private key data (via cascade)
4. Automation settings (via cascade)
5. Delegation data (via cascade)
6. The user account
7. Cognito identity (deleted/deactivated)

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token (JWT) from OAuth2 login
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message confirming account deletion
</ResponseField>

<ResponseField name="user_id" type="integer">
  The ID of the deleted user
</ResponseField>

<ResponseField name="deleted_at" type="string">
  ISO 8601 timestamp of deletion
</ResponseField>

<ResponseField name="deleted_data" type="object">
  Summary of data that was deleted
  
  <Expandable title="Deleted Data Object">
    <ResponseField name="api_keys" type="integer">
      Number of API keys deleted
    </ResponseField>
    
    <ResponseField name="sessions" type="integer">
      Number of sessions deleted
    </ResponseField>
    
    <ResponseField name="private_key" type="boolean">
      Whether a private key was deleted
    </ResponseField>
    
    <ResponseField name="automation_settings" type="boolean">
      Whether automation settings were deleted
    </ResponseField>
    
    <ResponseField name="delegations" type="integer">
      Number of delegations deleted
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="cognito_deletion" type="object">
  Information about Cognito identity deletion
  
  <Expandable title="Cognito Deletion Object">
    <ResponseField name="success" type="boolean">
      Whether Cognito deletion was successful
    </ResponseField>
    
    <ResponseField name="cognito_user_id" type="string">
      The Cognito user ID that was deleted
    </ResponseField>
    
    <ResponseField name="message" type="string">
      Status message from Cognito deletion
    </ResponseField>
  </Expandable>
</ResponseField>

<Warning>
  **IRREVERSIBLE ACTION**: This permanently deletes your account and all associated data. This action cannot be undone. Make sure to:
  - Export any data you need before deletion
  - Close any active sessions
  - Save your API keys if needed for records
  - Understand that all blockchain delegations will be revoked
</Warning>

<Warning>
  **Active Sessions**: Any active blockchain sessions will be terminated immediately upon account deletion. This may result in loss of unused session time.
</Warning>

<Info>
  **GDPR Compliance**: This endpoint ensures full compliance with data deletion requirements under GDPR and similar privacy regulations by removing all personal data from both the application database and the authentication provider (Cognito).
</Info>

<Note>
  After account deletion, you will be immediately logged out and your JWT token will be invalidated. You will need to create a new account if you wish to use the service again.
</Note>
