Root endpoint returning basic API information.

Returns metadata about the Morpheus API Gateway including version information and available resources.

### Response

<ResponseField name="name" type="string">
  API name
</ResponseField>

<ResponseField name="version" type="string">
  Current API version
</ResponseField>

<ResponseField name="description" type="string">
  API description
</ResponseField>

<ResponseField name="documentation" type="string">
  URL to API documentation
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.mor.org/
```

```javascript JavaScript
const response = await fetch('https://api.mor.org/');
const info = await response.json();
```

```python Python
import requests

response = requests.get('https://api.mor.org/')
info = response.json()
```
</RequestExample>

<ResponseExample>
```json 200 Response
{
  "name": "Morpheus API Gateway",
  "version": "v1.1.1",
  "description": "API Gateway connecting Web2 clients to the Morpheus-Lumerin AI Marketplace",
  "documentation": "https://docs.mor.org"
}
```
</ResponseExample>

<Info>
  **API Discovery**: This endpoint is useful for API discovery and version checking. It provides a quick way to verify the API is accessible and learn about its current version.
</Info>

<Note>
  This endpoint does not require authentication and can be called by anyone to check API availability.
</Note>
