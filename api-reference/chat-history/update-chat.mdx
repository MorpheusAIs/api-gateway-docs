---
title: "Update Chat"
api: "PUT /api/v1/chat-history/chats/{chat_id}"
description: "Update chat title"
---

Update chat title.

Allows you to rename a chat conversation. This is useful for organizing and categorizing your chats.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token (JWT) from OAuth2 login
</ParamField>

### Path Parameters

<ParamField path="chat_id" type="string" required>
  The ID of the chat to update
</ParamField>

### Body

<ParamField body="title" type="string" required>
  Updated chat title (max 200 characters)
</ParamField>

### Response

<ResponseField name="id" type="string">
  Unique identifier for the chat
</ResponseField>

<ResponseField name="title" type="string">
  Updated chat title
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp when created
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp when last updated
</ResponseField>

<ResponseField name="message_count" type="integer">
  Number of messages in the chat
</ResponseField>

<RequestExample>
```bash cURL
curl --request PUT \
  --url https://api.mor.org/api/v1/chat-history/chats/chat_123abc \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "title": "Renamed Conversation"
  }'
```

```javascript JavaScript
const response = await fetch(
  'https://api.mor.org/api/v1/chat-history/chats/chat_123abc',
  {
    method: 'PUT',
    headers: {
      'Authorization': 'Bearer YOUR_JWT_TOKEN',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title: 'Renamed Conversation'
    })
  }
);

const updatedChat = await response.json();
```

```python Python
import requests

response = requests.put(
    'https://api.mor.org/api/v1/chat-history/chats/chat_123abc',
    headers={
        'Authorization': 'Bearer YOUR_JWT_TOKEN',
        'Content-Type': 'application/json'
    },
    json={'title': 'Renamed Conversation'}
)

updated_chat = response.json()
```
