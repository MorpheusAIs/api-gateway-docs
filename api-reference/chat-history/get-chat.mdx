title: "Get Chat"
api: "GET /api/v1/chat-history/chats/{chat_id}"
description: "Get a specific chat with all messages"
---

Get a specific chat with all messages.

Returns the complete chat conversation including all messages in chronological order.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token (JWT) from OAuth2 login
</ParamField>

### Path Parameters

<ParamField path="chat_id" type="string" required>
  The ID of the chat to retrieve
</ParamField>

### Response

<ResponseField name="id" type="string">
  Unique identifier for the chat
</ResponseField>

<ResponseField name="title" type="string">
  The chat title
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp when created
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp when last updated
</ResponseField>

<ResponseField name="messages" type="array">
  Array of message objects in chronological order
  
  <Expandable title="Message Object">
    <ResponseField name="id" type="string">
      Unique message identifier
    </ResponseField>
    
    <ResponseField name="role" type="string">
      Message role: `user` or `assistant`
    </ResponseField>
    
    <ResponseField name="content" type="string">
      Message content
    </ResponseField>
    
    <ResponseField name="sequence" type="integer">
      Message order in conversation
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
    
    <ResponseField name="tokens" type="integer">
      Token count (if available)
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.mor.org/api/v1/chat-history/chats/chat_123abc \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```

```javascript JavaScript
const response = await fetch(
  'https://api.mor.org/api/v1/chat-history/chats/chat_123abc',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer YOUR_JWT_TOKEN'
    }
  }
);

const chatWithMessages = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.mor.org/api/v1/chat-history/chats/chat_123abc',
    headers={'Authorization': 'Bearer YOUR_JWT_TOKEN'}
)

chat_with_messages = response.json()
```

<Note>
