---
title: "Delete Chat"
api: "DELETE /api/v1/chat-history/chats/{chat_id}"
description: "Delete or archive a chat"
---

Delete or archive a chat.

By default, chats are archived rather than permanently deleted. Use the `archive_only` parameter to control this behavior.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token (JWT) from OAuth2 login
</ParamField>

### Path Parameters

<ParamField path="chat_id" type="string" required>
  The ID of the chat to delete
</ParamField>

### Query Parameters

<ParamField query="archive_only" type="boolean" default={true}>
  If true, archives the chat. If false, permanently deletes it.
</ParamField>

### Response

Returns `204 No Content` on successful deletion.

<RequestExample>
```bash cURL - Archive
curl --request DELETE \
  --url 'https://api.mor.org/api/v1/chat-history/chats/chat_123abc?archive_only=true' \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```

```bash cURL - Permanent Delete
curl --request DELETE \
  --url 'https://api.mor.org/api/v1/chat-history/chats/chat_123abc?archive_only=false' \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```

```javascript JavaScript
// Archive (default)
const response = await fetch(
  'https://api.mor.org/api/v1/chat-history/chats/chat_123abc?archive_only=true',
  {
    method: 'DELETE',
    headers: {
      'Authorization': 'Bearer YOUR_JWT_TOKEN'
    }
  }
);

// Permanent delete
const response = await fetch(
  'https://api.mor.org/api/v1/chat-history/chats/chat_123abc?archive_only=false',
  {
    method: 'DELETE',
    headers: {
      'Authorization': 'Bearer YOUR_JWT_TOKEN'
    }
  }
);
```

```python Python
import requests

# Archive (default)
response = requests.delete(
    'https://api.mor.org/api/v1/chat-history/chats/chat_123abc',
    params={'archive_only': True},
    headers={'Authorization': 'Bearer YOUR_JWT_TOKEN'}
)

# Permanent delete
response = requests.delete(
    'https://api.mor.org/api/v1/chat-history/chats/chat_123abc',
    params={'archive_only': False},
    headers={'Authorization': 'Bearer YOUR_JWT_TOKEN'}
)
```
</RequestExample>

<Info>
  **Archive vs Delete**: 
  - **Archive** (`archive_only=true`): Hides the chat from your list but keeps the data for potential recovery
  - **Permanent Delete** (`archive_only=false`): Irreversibly removes the chat and all associated messages
</Info>

<Warning>
  **Irreversible**: When `archive_only=false`, the chat and all its messages are permanently deleted. This action cannot be undone.
</Warning>

<Tip>
  Use archiving as the default to prevent accidental data loss. Only use permanent deletion when you're certain you won't need the conversation history.
</Tip>
